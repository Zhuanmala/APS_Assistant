# APS陪练助手（MVP）

这是一个可本地直接运行的APS陪练产品原型，面向准备去德国留学的中国学生。

## 已实现功能

1. 路径诊断：根据基础信息生成 `Interview / TestAS` 建议与风险提示。
2. 模拟面谈：10题自适应问答，按课程真实性、理论理解、实践应用、表达清晰度评分。
3. 语音陪练：支持题目朗读、语音回答转文字、语音提交评分。
4. 自动保存记录：每题回答（文本/语音）自动保存到本地存储。
5. 材料雷达：按路径生成材料清单，勾选后输出缺失风险。
6. 复盘作战卡：输出路径建议、薄弱项TOP3、关键缺口与7天训练计划。
7. 导出复盘PDF：一键导出当前会话的摘要与答题记录。
8. Gemini接入：可配置 API Key 与模型，面谈题目生成、评分、追问、纠偏均由 Gemini 负责。
9. 科目绑定：可在打勾框中勾选已学科目（含设计类），Gemini会据此出题与追问。

## 如何运行

- 方式A：直接双击打开 `index.html`
- 方式B：在 VS Code 里右键 `index.html`，使用浏览器打开

## Gemini配置

1. 在第2步“模拟面谈”中找到“Gemini教练配置”。
2. 粘贴你的 Gemini API Key，并选择模型。
3. 点击“保存配置”，再点“测试连通”。
4. 连通成功后即可开始模拟；未配置Key时，面谈不会启动（全Gemini模式）。
